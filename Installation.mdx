#### Requirements:

- gcc 10\+ or clang 10\+
- x86\_64 os
- termcolor lib

Installing Flame via git:

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame
make -j$(nproc)
sudo make install
```

Installing Shiver(build system) via git:

#### if already did what above

```sh
cd shiver
make
```

#### else

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame/shiver
make
```
