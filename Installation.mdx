#### Requirements:

- GCC 10\+ or Clang 10\+
- x86\_64 OS

#### Installing Flame via git:

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame
make -j$(nproc)
make install 
cd shiver 
make
make install 
```

#### Installing via AUR (Arch Linux):

```sh
yay -S flamelang-git
```
