#### Requirements:

- GCC 10\+ or Clang 10\+
- x86\_64 OS

#### Installing Flame via git:

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame
make -j$(nproc)
make install # installing Flame
cd Shiver 
make
make install # installing Shiver(build system)
```

#### Installing via AUR (Arch Linux):

```sh
yay -S flamelang-git
```
