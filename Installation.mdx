#### Requirements:

- GCC 10\+ or clang 10\+
- x86\_64 OS

#### Installing Flame via git:

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame
make -j$(nproc)
sudo make install # installing Flame
cd Shiver 
make
make install # I
```

#### Installing Shiver(build system) via git:

if already did what above

```sh
cd shiver
make
```

otherwise

```sh
git clone https://github.com/TheFlameLang/Flame.git
cd Flame/shiver
make
```
